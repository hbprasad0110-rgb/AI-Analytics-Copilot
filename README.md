# AI Analytics Copilot (Free, Local)

An end-to-end **AI Analytics Copilot** that converts natural language questions into SQL queries and visual insights.

Built using:
- PostgreSQL (local database)
- Ollama (local LLM)
- FastAPI (backend)
- Streamlit (frontend)

---

## Features
- Ask questions in plain English
- Automatic SQL generation
- Executes queries on PostgreSQL
- Displays tabular results and charts
- Fully local — no paid cloud services

---

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- Ollama (LLM)
- Streamlit
